# PT1000Control
Author: Janine Menne

## Description
C++ class which uses a Keithley Multimeter to read out a PT1000 element.

## Requirements to run terminal program

Availability of the following libraries (author: Janine Menne):  
* [MultiMeter][2]  
* [LogFileDev][4]  
* [gpib][5]
